Section 3

6 Font Sizes — Inline Styling

Same text, six different CSS size units.

  • The quick brown fox jumps over the lazy dog. 12px — Good for fine print, footnotes, or legal disclaimers where size must never change regardless of user preferences.
  • The quick brown fox jumps over the lazy dog. 1rem — Relative to the root (<html>) font size. Best for body text — it respects the user's browser font-size setting.
  • The quick brown fox jumps over the lazy dog. 1.25em — Relative to the parent element's font size. Useful inside reusable components where the component should scale with its container.
  • The quick brown fox jumps over the lazy dog. 120% — Percentage of the parent's font size.
  • The quick brown fox jumps over the lazy dog. 18pt — Absolute point unit. Meaningful for print stylesheets where you need precise physical dimensions.
  • The quick brown fox jumps over the lazy dog. 3vw — Relative to the viewport width. Great for hero headings that should scale fluidly with the browser window.

Section 4

4 CSS Positioning Options

Each colored box below demonstrates one positioning value.

static — default flow The default for every element. Follows the normal document flow; top/left/right/bottom have no effect. Use it when you need an element to simply appear where it belongs in the HTML.
relative — nudged 12px↓ 20px→ Stays in the normal flow (its space is reserved) but visually shifts by top/left. Use it to fine-tune position or to create a positioning context for absolute children.
sticky — stays at top:80px while scrolling Behaves like relative until the user scrolls past the threshold, then sticks. Perfect for section headers or sidebars that should stay visible inside a scrolling container.
absolute — bottom-right corner Removed from normal flow and placed precisely relative to the nearest positioned ancestor. Use for overlays, badges, tooltips, or decorative elements anchored to a card corner.
